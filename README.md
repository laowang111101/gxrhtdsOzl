## 前言

随着我国社会的发展，急救常识的重要性日益凸显。在突发情况下，正确的急救方法可以拯救生命，减少伤害。为了普及急救知识，提高公众的急救技能，我们开发了一款基于微信小程序的急救常识学习系统。本文将详细介绍该项目的相关内容。

## 内容介绍

本项目旨在为用户提供一个便捷、实用的急救常识学习平台。系统主要包括以下几个模块：

1. **课程学习**：用户可以通过课程学习模块了解各种常见病症的急救方法，如心脏病发作、中暑、溺水等。
2. **在线测试**：通过在线测试模块，用户可以检验自己的急救知识掌握程度，从而更好地巩固所学内容。
3. **急救手册**：提供一份详尽的急救手册，方便用户随时查阅。
4. **互动交流**：用户可以在互动交流模块提问、分享经验，共同进步。

## 技术介绍

本项目采用以下技术实现：

- **语言**：Java
- **使用框架**：Spring Boot
- **前端技术**：JS、Vue、css3
- **开发工具**：IDEA/Eclipse
- **数据库**：MySQL 5.7/8.0
- **数据库管理工具**：phpstudy/Navicat
- **JDK版本**：jdk1.8
- **Maven**：apache-maven 3.8.1-bin
- **前端环境**：Node.Js 12\14\16
